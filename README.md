Center Toys 🧸 - Sistema de Cadastro
Sistema desktop desenvolvido em Java para cadastro, consulta, edição e exclusão de produtos 
(brinquedos), com integração ao banco de dados MySQL.

Tecnologias utilizadas
- Java (Swing - Interface gráfica)
- MySQL (Banco de dados)
- JDBC (Conexão com banco)
- Eclipse IDE

Funcionalidades
✅ Cadastrar produtos
🔍 Buscar produtos pelo nome
✏️ Editar informações do produto
❌ Excluir produtos
🖼️ Adicionar e exibir imagem do produto
📂 Listagem dinâmica conforme digitação
